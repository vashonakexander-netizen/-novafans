# Easiest Way to Scrape Movies - Browse.ai

## Why Browse.ai is Easiest

✅ **No coding needed** - Just point and click
✅ **Handles login automatically** - Login once, it remembers
✅ **No cookie management** - It saves your session
✅ **Visual interface** - See what it's doing
✅ **Works immediately** - No setup headaches

## Quick Setup (5 minutes)

### Step 1: Go to Browse.ai
Open: https://dashboard.browse.ai

### Step 2: Create Robot
1. Click **"Create Robot"** or **"New Robot"**
2. Select **"Custom Robot"**
3. Paste your movie URL: `https://toflx.com/movies/sinners-2025`
4. Click **"Create"**

### Step 3: Record Session (IMPORTANT!)
1. **Enable "Record Session Cookies"** ✅
2. Browse.ai will open a browser window
3. **Login to toflx.com** in that browser
4. Browse.ai automatically saves your login session!

### Step 4: Configure What to Extract
1. Wait for the page to load
2. Tell Browse.ai what to extract:
   - **Video URLs** - Click on video player, select "Extract URL"
   - **Images** - Click on images, select "Extract"
   - **Movie title** - Click on title, select "Extract text"

### Step 5: Save & Run
1. Click **"Save Robot"**
2. Click **"Run"**
3. Browse.ai will:
   - Login automatically (using saved session)
   - Wait for content to load
   - Extract all the data
4. **Get results** from dashboard or download as JSON/CSV

## That's It!

No coding, no cookies, no setup - just point, click, and get your data!

## Getting Results

### Option 1: Dashboard
- View results in Browse.ai dashboard
- Download as JSON or CSV
- See all extracted URLs

### Option 2: API (Optional)
If you want to automate it later:
1. Get your API key from Browse.ai settings
2. Get your Robot ID from the robot page
3. Use the API to run it programmatically
